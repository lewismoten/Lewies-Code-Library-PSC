Image Drop Demonstration

This script allows you to specify an image to fall down the entire height of the web browser. Images drop at random speeds. The number of images dropping may be specified as well. This particular script demonstrates how to drop an image related to the current month. This gives off an interesting effect.

JanuarySnow Flakes
FebruaryHearts (Valentines Day)
MarchClovers (St. Patricks Day)
AprilRain Drops (April showers bring may flowers)
MayFlowers
JuneSmilies
JulyUnited States Flags (Independance Day July 4th)
AugustRays of Sun
SeptemberSmilies
OctoberSpider Webs (Halloween)
NovemberTurkeys (Thanksgiving)
DecemberPine Trees (Christmas)
